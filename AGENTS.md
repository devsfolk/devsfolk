# Project Rules & Constraints

## Mobile Layout Stability
- **Locked Mobile Views**: The mobile version of the Dashboard and the Storefront are considered finalized and "locked".
- **Permission Required**: Do NOT modify or touch the mobile-specific styles, layouts, or responsiveness logic for these sections without explicit permission from the user. 
- **Preserve Current State**: Any new features or changes being implemented must ensure they do not break or alter the existing mobile experience.
